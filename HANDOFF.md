# Handoff Notes

This file is how agents pass work to each other. When another agent has a task for you, they write it here so you get the full context without Taya having to re-explain everything.

## How It Works
- Another agent writes a task below with context
- When you start your session, check this file first
- Once you finish a handoff task, move it to "Completed" at the bottom
- If you need to hand something OFF to another agent, write to THEIR HANDOFF.md (ask Taya to save it there, or reference what to tell them)
